# 🌿 Air Quality Detection & Advisory System

A professional Air Quality Monitoring project that fetches real-time AQI data,
performs analysis and predictions, and provides health advisories.

## 🚀 Features
- Real-time Air Quality Index (AQI)
- API-based data fetching
- Machine Learning prediction (optional)
- Health advisory system
- Clean & modular structure

## 🧠 Tech Stack
- Python
- Machine Learning
- OpenWeather API
- Streamlit / Flask

## 📁 Project Structure
air-quality-detection/
├── src/
├── requirements.txt
├── README.md

## ⚙️ Installation
git clone https://github.com/Vikra444/air-quality-detection.git
cd air-quality-detection
pip install -r requirements.txt
